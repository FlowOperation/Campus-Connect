# Community Safety

## Safety Model
- Prevent abuse through proactive controls.
- Detect harmful behavior with automation signals.
- Respond quickly through moderator tooling.
- Learn through incident retrospectives.

## Protective Features
- User-level block and mute.
- Content and message reporting.
- Progressive sanctions for repeat violations.
- Campus-aware policy enforcement.

## Incident Response
- Severity levels with response SLA.
- Incident commander assignment for major events.
- Post-incident review with remediation tracking.

## Safety Metrics
- Abuse report rate per 1k active users.
- Time to first moderation action.
- Repeat offender rate.
- False positive and appeal reversal rates.
